# Horizontal barchart

D3 implementation of horizontal barchar

**Features**:
- Object oriented approach
- Responsive
- Resizable

**Default options**:
```javascript
{
    'margin': {'top': 40, 'right': 30, 'bottom': 50, 'left': 40},
    'key': 'key',
    'label': 'date',
    'color': 'steelblue',
    'title': false,
    'source': false,
}
```
